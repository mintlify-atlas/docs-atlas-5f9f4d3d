</Note>

<Update label="2024-09-19" tags={["Foundation"]}>
  ## Initial data model and persistence

  Established the core domain objects and the JSON persistence layer that underpins the application.

  - Implemented the `Food` object with `id`, `price` (€/kg), and `energy` (kcal/100g) properties.
  - Implemented the `Recipe` object containing a list of `Food` items with quantities, preparation time, and comments.
  - Implemented loading and saving data to JSON databases, enabling the food and recipe catalogs to persist between sessions.
</Update>
